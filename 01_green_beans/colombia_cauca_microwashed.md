## 生豆特征分析

### 处理法特性 — 微生物水洗

微生物水洗（接菌水洗）是近年流行的新式处理法，介于传统水洗和厌氧处理之间：

- 在水洗发酵阶段引入**特定菌株**（酵母/乳酸菌）控制发酵
- 比传统水洗**糖分含量更高**（菌群产糖代谢）
- 比厌氧日晒**风味更干净**（保留水洗的明亮酸）
- 杯测特征：复杂果酸 + 发酵感 + 干净尾韵，常带「奶油感/酒韵」
- 烘焙特征：**梅纳德反应底物丰富**（接近日晒），但**含水率和细胞结构更接近水洗**

> 关键含义：烘焙时按「水洗豆温和升温节奏」处理细胞结构，但需要**足够的梅纳德时间**让发酵糖类充分参与反应（否则风味发挥不出）。

### 密度分析 — 796 g/L（中低密度）

实测密度 796 g/L 处于中低密度区间：

| 参考豆种 | 密度 | 实测 FC 温度 |
|---------|---:|---:|
| Catimor 云南 1650m 日晒 | 759 g/L | ~190-196°C |
| **COL Cauca（本豆）** | **796 g/L** | **预测 192-196°C** |
| RWA 红波旁 1800-2000m 日晒 | 817 g/L | 188.7-196.3°C（实测） |
| ETH 罕贝拉 1900-2300m 水洗 | 892 g/L | ~186-197°C |

低密度 → 热传导效率高 → 升温快 → **FC 来得偏早**。预测实际 FC 落在 **192-196°C** 区间，比 expect_fc 模板值 206°C 偏低约 10-14°C。

### 陈化评估 — 中度（3+ 个月）

| 维度 | 评估 |
|------|------|
| 储存时长 | 3+ 个月（远短于 COL Chiloso 的 8 个月） |
| 含水率漂移 | 标称 11.2% → 实际估 10.7-11.0%（漂移轻微） |
| 风味损失 | 极低（顶香、果酸基本保留） |
| 烘焙影响 | 几乎可忽略，按新鲜豆方案设计即可 |

**结论：** 不属于「严重陈化豆」，无需 Chiloso 那样的激进补偿（preheat 大幅降低、Level 上调等）。

### ⚠️ 关键风险点（用户明确警示）

用户要求：**避免烘焙不熟 / 夹生 / 半生**。

历史失败案例参考（COL Chiloso V2）：
- 自定义 roast_profile 贝塞尔曲线格式错误 → 固件截断
- ROR ≈ 31°C/min 直线斜坡，烘焙仅 283s 即被 end_ratio 终止
- 失重仅 3.3%，豆芯未熟

**本配置的预防措施：**
1. ✅ **不自定义 roast_profile** — 完全使用官方 F-WSH 1500-2200 验证曲线
2. ✅ **end_ratio 保持 0.75** — 不下调到 0.6/0.7
3. ✅ **preheat_power ≥ 815W** — 确保前段热量充足
4. ✅ **expect_fc 留 5-7°C 安全余量** — 防止 PID 提前进入 FC 模式致停滞
5. ✅ **Level 推荐 2.0（207°C），不低于 1.5（205°C）** — 出豆温度有保障

## 烘焙建议

- 目标烘焙度：**浅度烘焙**
- ⛔ Level 1.0 (203°C)：不推荐 — 发展不足风险
- Level 1.5 (205°C)：最浅安全档（明亮花果酸）
- **Level 2.0 (207°C)：⭐ 推荐 — 平衡浅焙**
- Level 2.5 (209°C)：浅中焙备选（突出微生物水洗的发酵复杂度）
- Level 3.0 (211°C)：中焙（如首炉杯测偏青草/麦芽感时上调）
- C40 手冲研磨度：22-26 格
- 养豆期：7-14 天
- 建议投豆量：120g

## 首炉操作建议

1. 投豆 **120g**，选择 **Level 2.0**（207°C 出豆）
2. **冷启动**（chamber ambient ≤ 25°C），不要 B2B
3. 烘焙前记录：ambient、市电电压、距上炉时间
4. 重点观察：
   - FC 信号（预计 ~192-196°C，比 expect_fc 偏低 5-10°C 正常）
   - 中段 ROR 是否平稳（无负 ROR 才是健康标志）
   - 出豆温度是否命中 207±0.3°C
5. **首炉必须称重**（生豆 / 熟豆），计算失重率（健康范围 10-13%）
6. 养豆 7 天后杯测，重点关注：
   - 是否有夹生感（青草/麦芽/纸质）→ 如有则下一炉 Level→2.5
   - 是否过度发展（苦尾/烟感）→ 如有则下一炉 Level→1.5
   - 微生物水洗的发酵复杂度是否展现
