## 八、问题诊断

### 1. log0082 严重负 ROR (核心问题)
- 位置: 267-276s / 180°C
- 严重程度: ROR -7.43（V3 最差仅 -0.28）
- 推测原因: expect_fc=192 + back2back 热状态 → PID 在 180°C 过早减速 → 停滞 → 过度反弹
- 建议: 将 expect_fc 回调至 193-194，或仅在非连续烘焙条件下使用 V4

### 2. 发展比偏高 (23-24%)
- V3 为 20.3%，V4 目标 17-20%
- 实际发展比反而增加，原因：FC 温度似乎降至 187-189°C（低于 V3 的 190°C）
- expect_fc=192 可能导致 PID 提前进入 FC 减速模式，使实际 FC 发生在更低温度
- 发展段从 ~187°C 到 ~203°C 跨度 16°C（V3 从 190°C 到 203.3°C 跨度 13°C）

### 3. 中段偏冷依然存在
- -6.1°C 最大偏冷（V3 为 -5.3°C）
- 问题持续出现在 155-165s / 144-146°C 区域
- 需考虑：中段曲线节点是否需要进一步放缓

## 九、建议

### V5 调整方向
1. **expect_fc 回调至 194**: 192 过低导致 PID 过早减速和负 ROR 风险
2. **终点微调 395s/203°C**: 再降 1°C 逼近甜感峰值（081 的 203.2 仍略高）
3. **中段斜率继续放缓**: 195s/166°C → 200s/165°C，减缓 PID 追踪压力
4. **连续烘焙注意**: back2back 时 PID 表现差异显著，如需连续烘焙考虑单独配置
