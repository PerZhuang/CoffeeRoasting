## 关键观察

1. **「不夹生」目标完美达成** — 用户最关注的风险（夹生/烘焙不熟）零出现，验证 V1 五重防护策略（官方模板曲线 + end_ratio 0.75 + preheat 820W + expect_fc 200°C + Level 2.0 不下探）有效。

2. **杯测时机存疑** — 烘焙日 = 杯测日（养豆 0 天）。CO2 排气未稳定可能扭曲萃取效率（萃取率 17.81% 偏低）和风味判断（烟感可能虚高）。**强烈建议养豆 5-7 天后做对照杯测**。

3. **失重率 10.33% 是浅焙下限** — 健康区间 10-14%，本炉踩在下沿。如果 V2 想强化果酸甜厚度，可微调（见 V2 建议）。

4. **微生物水洗的"水果酸糖"特征已展现** — 豆商描述完全兑现，无需进一步追求"更高复杂度"。

## 下一步建议

### 立即操作（无需新烘焙）
1. **养豆 5-7 天后再冲一杯对照**（粉量保持 22g，水温 92°C，但建议调粗 1-2 格 + 水比降至 1:14）
2. 观察烟草味是否随养豆减弱 — 如减弱则确认是 D 假设（CO2 残留），无需 V2
3. 同时尝试 1:14 + 略粗研磨，看 TDS 是否降至 1.30% 区间

### V2 烘焙方向（若养豆后烟草味仍存）

| 方向 | 调整 | 适用场景 |
|------|------|---------|
| **方向 1（推荐）：维持 V1，仅微调研磨/水比** | 不改 .kpro | 烟草味是个人偏好范畴而非缺陷 |
| 方向 2：V2 降至 Level 1.5 (205°C) | recommended_level: 2.0 → 1.5 | 想突出果酸花香，去除任何烟感 |
| 方向 3：V2 升至 Level 2.5 (209°C) | recommended_level: 2.0 → 2.5 | 想要更厚实的酱体（参考 RWA 思路）|
| ⛔ 不推荐：V2 缩短发展时间 | end_ratio 下调 | 风险：触发夹生 |

### 监控点
- 下一炉（V1 第二炉）的 FC 温度是否仍在 196-198°C → 验证 V1 一致性
- 失重率是否能稳定在 10.5-11% → 用作 V2 决策依据
