### 设计思路

**基础模板选择**：F-NAT 1500-2200（日晒处理 + 海拔 1800-2000m 落在此范围中段）

**关键考量因素**：
1. **日晒处理**：含糖量高于水洗，前段热量需温和避免焦化。preheat 从模板 850W 降至 820W。
2. **中等密度 817g/L**：介于 Catimor(759) 和 ETH(892) 之间，不需要极端的热量设置。820W 平衡了日晒保护和中密度热需求。
3. **新鲜豆**：含水率 11.6% 正常，无需陈化补偿（与 COL Chiloso 的情况完全不同）。
4. **expect_fc=202°C**：汲取 COL V3 教训（200°C 过低致 PID 停滞），设稍高的 202°C 作为保守值。首炉验证实际 FC 后微调。

**关键发现：**

1. **零负 ROR** ✅ — Kd=3.10 + F-NAT 模板组合表现优秀，全程无 PID 停滞
2. **FC 偏早**：ROR 从 18.4 下降至 6.4（305-309s），对应温度 ~194°C，比 expect_fc=202°C 早 8°C。ROR 先降后弹（6.4→15.0→10），呈典型 FC 响应。日晒豆 FC 偏早符合预期（高糖加速反应）
3. **发展比 19.5%**：因 FC 偏早（194°C），从 FC 到出豆的实际发展段更长，导致发展比偏高。若 FC 取 197°C，发展比约 17.5%
4. **出豆稳定**：末段 ROR 平稳维持 11-12°C/min，无冲刺无停滞，出豆 207.5°C 精确命中 Level 2.0
5. **Profile 跟踪良好**：中段偏冷 -3°C（可接受），末段反超 +1-2°C
6. **失重率 10.8%**：正常浅中焙范围

**V2 调整方向（如需优化）：**
- expect_fc: 202→198°C（对齐实际 FC ~194°C，但需谨慎避免 COL V3 的过低问题）
- 或保持现状——当前 PID 表现已经很好，零负 ROR
- 梅纳德 ROR 17.5 可接受，如需放缓可微降 preheat 至 810W

### 首炉操作建议（归档）

1. 投豆 **120g**，选择 **Level 2.0**
2. 注意中段（120-250s）是否偏冷（820W 为保守值）
3. 如中段偏冷 >5°C，V2 可考虑 preheat 上调至 830-840W
4. 记录出豆重量，注意 FC 信号（预计 ~198-203°C 区间）
5. 日晒豆 FC 信号通常比水洗更清晰（糖分辅助反应）
6. 养豆 7-14 天后杯测
