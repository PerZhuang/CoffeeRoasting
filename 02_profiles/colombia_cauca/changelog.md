# 哥伦比亚 考卡山 微生物水洗 — 配置变更日志

## V1 (2026-05-16) — 待首炉测试
**文件：** `Colombia_Cauca_MicroWashed_Filter_v1.kpro`
**基础：** Kaffelogic 官方 F-WSH 1500-2200 (SeanBrews v2.2) 模板
**投豆量：** 120g

### 生豆背景

- 哥伦比亚考卡山产区，海拔 **1895-1950m**（实测）
- 微生物水洗处理（接菌发酵水洗）
- 含水率 11.2%（标称）
- 密度 **796 g/L**（实测，中低密度区间）
- 储存时长 3+ 个月（中度陈化）
- 用户明确要求：**避免烘焙不熟 / 夹生**

> **密度-海拔异常说明**：1895-1950m 高海拔豆典型密度应在 830-880 g/L，本豆实测 796 g/L 偏低。可能原因：品种（卡杜拉/哥伦比亚等低密度品种）+ 微生物发酵 + 储存失水。**烘焙以密度为主要热传导依据**（非海拔），故 V1 参数不受海拔修正影响。

### 设计思路

**基础模板选择：F-WSH 1500-2200**
- 考卡山典型海拔 1700-2100m 落在模板上半段
- 水洗处理直接匹配 F-WSH 系列
- 已通过 COL Chiloso V1（log0099）和 RWA 等多支豆验证

**关键考量因素**

**1. 密度 796 g/L 的物理含义**

豆子热传导效率较高（低于 RWA 817 但高于 Catimor 759）。

| 参考豆 | 密度 | 实测 FC 温度 |
|---------|---:|---:|
| Catimor V5 | 759 g/L | 190-196°C |
| **本豆（预测）** | **796 g/L** | **192-196°C** |
| RWA V1 | 817 g/L | 188.7-196.3°C |

→ 预测实际 FC 在 192-196°C 区间，与 RWA 高度相似。

**2. 微生物水洗的烘焙特性**

- 糖分含量高于传统水洗（菌群代谢产糖）
- 烘焙特性介于水洗和日晒之间
- 梅纳德反应底物丰富 → 需要**足够发展时间**让发酵糖参与反应
- 浅焙下限不能太低，否则发酵复杂度无法展现

**3. 陈化程度评估（中度）**

| 维度 | COL Chiloso（参考） | 本豆 |
|------|---:|---:|
| 储存时长 | 8 个月 | 3+ 个月 |
| 陈化等级 | 严重 | 中度 |
| 含水率漂移 | 12% → ~10% | 11.2% → ~10.7-11% |
| 风味损失 | 显著 | 轻微 |
| 推荐补偿 | 激进（preheat 790W） | 温和（preheat 820W）|

**结论：** 本豆陈化远轻于 Chiloso，按新鲜豆设计即可，仅 preheat 略低于模板（820W vs 840W）。

**4. 防夹生的工程保障**

| 风险点 | V1 防护措施 |
|------|------|
| 自定义曲线格式错误 | ✅ **完全使用官方模板曲线** |
| end_ratio 过低致提前终止 | ✅ 保持 0.75 标准值 |
| 预热不足致前段冷启动 | ✅ preheat 820W（不低于 815W）|
| expect_fc 过低致 PID 停滞 | ✅ 200°C，留 5-7°C 安全余量 |
| 出豆温度过低 | ✅ Level 2.0 = 207°C，不推荐 Level 1.0 |

### 首炉操作清单

烘焙前确认（参考 SOP_multibatch_roasting.md）：

- [ ] 冷启动（chamber ambient ≤ 25°C）
- [ ] 不要与其他烘焙背靠背
- [ ] 投豆 **120g**，选择 **Level 2.0**
- [ ] 记录烘焙前数据：ambient / 市电电压 / time_jump
- [ ] **必须称重**：生豆 120.0g、出豆 ?g、失重率 ?%

烘焙中观察：

- [ ] 中段（120-250s）profile 跟踪是否正常（mean_temp 偏差 < 5°C）
- [ ] FC 信号（预计 ~193°C / 300-310s）— 听爆裂声 + 看 ROR 急降
- [ ] FC 后 ROR 是否健康反弹（避免类似 log0105 的 PID 停滞）
- [ ] 出豆温度命中 207±0.5°C

烘焙后（养豆 7 天再杯测）：

- [ ] 是否有**夹生感**（青草/麦芽/纸质味）→ 下一炉升至 Level 2.5
- [ ] 是否过度发展（烟感/苦尾）→ 下一炉降至 Level 1.5
- [ ] 微生物水洗的发酵复杂度（奶油感/酒韵/果脯）是否展现

### 风险评估

| 风险 | 等级 | 缓解措施 |
|------|:---:|------|
| 夹生 / 烘焙不熟 | 🟢 低 | 多重防护（见上方表格） |
