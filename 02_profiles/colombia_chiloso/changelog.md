### V2 失败回顾

⚠️ **V2 (log0100) 烘焙失败** — roast_profile 曲线格式错误导致：
- 曲线被固件截断（16 值 → 12 值），终点和终止符丢失
- 贝塞尔控制柄参数错误，曲线变为 ROR≈31°C/min 的直线斜坡
- 烘焙仅 283s(4:43) 即被 end_ratio 安全终止
- 出豆温度 198.1°C，失重仅 3.3%，豆子未烘熟

**教训：不要自行编写 roast_profile 贝塞尔曲线。使用官方模板或已验证的曲线形状。**

### V3 策略

恢复 V1 使用的 F-WSH 1500-2200 官方模板曲线（log0099 验证可正常工作），仅通过参数调整实现延长发展的目标。

### 设计思路

**基础模板选择**：F-WSH 1500-2200（海拔 1800-1900m 落在此范围中段）

**关键考量因素**：
1. **陈化补偿**：购入 8 个月，存放于北方干燥环境。含水率从 12% 估降至 ~10-10.5%。低含水率 → 降低 preheat（790W vs 模板 840W），避免脱水过快致表面焦化。
2. **密度未知**：基于海拔和品种估算 ~840-870g/L（中高密度）。保守设计，首炉观察实测响应。
3. **浅焙优先**：COE 品质豆，优先保留酸质透明度和品种风味。Level 2.0 = 204°C 作为目标。
4. **陈化风险对冲**：如果浅焙偏平淡/纸质，Level 2.5（206°C）作为备选。

### 首炉测试结果（log0099, 2026-04-20）

| 指标 | V1 设计目标 | log0099 实测 | 偏差 |
|------|:---:|:---:|:---:|
| 投豆量 | 120g | 130g | +10g |
| expect_fc | 204°C | 204°C | — |
| 实际 FC | ~198-204°C(预期) | ~197°C(中等信号) | 略低于预期范围 |
| FC 时间 | — | ~310s | — |
| 出豆温度 | ~204°C(Level 2.0) | 203.4°C | -0.6°C ✅ |
| 总时间 | — | 369.5s(6:09) | 偏快 |
| 发展时间 | — | ~60s | — |
| 发展比 | ~15%(目标) | ~16.2% | +1.2% ✅ |
| 出豆重量 | — | 待补充 | — |

**关键发现：**
1. **FC 命中偏差**：实际 FC ~197°C vs expect_fc 204°C，偏差 -7°C → V2 可考虑 expect_fc 下调至 200-201°C
2. **PID 稳定性**：负 ROR 5 次（最大 -1.4°C/min @203°C），集中在 FC 区域，幅度可接受
3. **FC 区域 ROR 抖动**：发展段 ROR 峰谷差 10.4，高于 Catimor V5 和 ETH V1 水平
4. **spot_temp 振荡**：振荡率 46%，motor_supply_noise=9；主因为流化床湍流，非 PID 缺陷
5. **总时间偏快**：369.5s，低于典型浅焙 6:30-7:00 范围，130g 投豆量下热量供应充足

**V2 调整方向：**
- expect_fc: 204→200°C（修正 FC 偏差，避免 PID 提前进入 FC 模式）
- 待杯测后决定是否调整 roast_levels 或 preheat
- 如杯测偏平淡/纸质，可尝试 Level 2.5（206°C）

### 首炉操作建议（归档）

1. 投豆 120g，选择 Level 2.0
2. 注意观察中段（120-250s）是否偏冷（预计可能偏冷 3-5°C，790W 偏保守）
3. 如中段偏冷 >5°C，V2 可考虑 preheat 上调至 810-820W
4. 记录出豆重量、全程注意 FC 信号（预计 ~198-204°C 区间）
5. 养豆 5-7 天后杯测

## V2 (2026-04-21) — 待测试

**基础：** V1 + log0099 首炉反馈 + 杯测数据

### V2 变更原因

V1 首炉杯测反馈：**闻香优秀但入口寡淡**。数据分析显示梅纳德阶段过快是主因。

### V2 关键参数变更

| 参数 | V1 | V2 | 变更理由 |
|------|-----|-----|---------|
| roast_profile | F-WSH 模板 4点曲线 | **自定义 6点曲线** | 添加中段控制点，延长梅纳德阶段 |
| expect_fc | 204°C | **200°C** | 实测 FC 197°C，缩小偏差至 3°C |
| recommended_level | 2.0 (204°C) | **2.5 (206°C)** | 陈化豆需更多发展补偿风味损失 |
| preheat_power | 790W | **810W** | V1 中段偏快但略冷，对齐 Catimor V5 验证值 |
| fan_profile | F-WSH 模板 | **自定义** | 匹配新曲线时间轴 |

### V2 曲线设计

**核心策略：延长梅纳德 + 提升发展度**

| 阶段 | V1 实测 | V2 目标 | 变化 |
|------|:---:|:---:|:---:|
| 脱水期 (→168°C) | 198s | ~210s | +12s |
| **梅纳德期 (168→FC)** | **106s** | **~130s** | **+24s** |
| 梅纳德 ROR | 15.4°C/min | ~13.8°C/min | -1.6 |
| 发展期 (FC→end) | 60s | ~80s | +20s |
| 发展比 | 16.2% | 18-20% | +2-4% |
| 总时间 | 369s | ~400-420s | +30-50s |
| 出豆温度 | 203.4°C | ~205-206°C | +2°C |

**roast_profile 曲线控制点：**

| 时间(s) | 温度(°C) | 阶段 |
|------:|------:|------|
| 0 | 20 | 起始 |
| 20 | 48 | 初始升温 |
| 50 | 85 | 脱水前段 |
| 210 | 168 | 转黄点（梅纳德起点） |
| 340 | 198 | 一爆区域（梅纳德终点，130s） |
| 420 | 206 | 出豆（Level 2.5） |

### V2 操作建议

1. 投豆 **120g**（回归标准量，便于与 V1 130g 对比）
2. 选择 **Level 2.5**（206°C）
3. 观察中段 ROR 是否降至 ~13-14°C/min（V1 为 15.4）
4. 注意 FC 信号时间（预计 ~330-340s 区间）
5. 记录出豆重量（V1 未记录）
6. 养豆 **5-7 天**后杯测，重点关注：醇厚度是否改善、甜感是否出现、是否出现过度发展苦味

### 风险评估

- **过度发展风险（低）**：Level 2.5 仍在浅焙范围，发展比目标 18-20% 对陈化豆适中
- **中段偏冷风险（低）**：preheat 从 790→810W，且自定义曲线中段斜率更缓
- **如仍寡淡**：可尝试 Level 3.0（208°C），或考虑此豆陈化已达风味极限
